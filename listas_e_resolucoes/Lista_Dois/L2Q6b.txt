Para gerar o AFD (Autômato Finito Determinístico) equivalente para a expressão regular (ab)*(ba)*, precisamos construir o NFA (Autômato Finito Não-Determinístico) correspondente e depois converter para um AFD.

Vamos começar criando o NFA:

1. Estado inicial: q0
2. Estado para reconhecer a sequência "ab": q1
3. Estado para reconhecer a sequência "ba": q2
4. Estado final: q3

Transições do NFA:

1. De q0 para q1 com "a": Representa a transição para o estado q1 ao encontrar o primeiro "a".
2. De q1 para q1 com "b": Representa a possibilidade de ocorrer a sequência "ab" novamente.
3. De q1 para q2 com "a": Representa a transição para o estado q2 ao encontrar "a" após a sequência "ab".
4. De q2 para q2 com "b": Representa a possibilidade de ocorrer a sequência "ba" novamente.
5. De q2 para q3 com "a": Representa a transição para o estado final q3 ao encontrar "a" após a sequência "ba".
6. De q3 para q2 com "b": Representa a transição para o estado q2 ao encontrar "b" após a sequência "ba".

Aqui está o NFA completo em forma de tabela:

```
| Estado Atual | Próximo Estado (a) | Próximo Estado (b) |
|--------------|--------------------|--------------------|
|     q0       |       q1           |       -            |
|     q1       |       q2           |       q1           |
|     q2       |       q3           |       q2           |
|     q3       |       -            |       q2           |
```

Agora, vamos converter esse NFA em um AFD usando o método de Subconjuntos:

1. Criar um novo estado inicial (q0) que será um conjunto dos estados alcançáveis a partir do estado inicial do NFA (q0).
2. Para cada conjunto de estados alcançável por uma transição a partir de um estado em q0, faça um novo estado do AFD.
3. Repita o passo 2 até que não haja novos estados a serem criados.

Vamos construir o AFD a partir do NFA:

1. Estado inicial: q0 (conjunto {q0, q1})
2. Novo estado para reconhecer a sequência "a": q1 (conjunto {q2, q3})
3. Novo estado para reconhecer a sequência "ab": q2 (conjunto {q1})
4. Novo estado para reconhecer a sequência "aba": q3 (conjunto {q2, q3})
5. Estado final: q4 (conjunto {q2})

Transições do AFD:

1. De q0 para q1 com "a": Representa a transição para o estado q1 (conjunto {q2, q3}) ao encontrar o primeiro "a".
2. De q1 para q2 com "b": Representa a transição para o estado q2 (conjunto {q1}) ao encontrar "b" após o "a".
3. De q2 para q3 com "a": Representa a transição para o estado q3 (conjunto {q2, q3}) ao encontrar "a" após "ab".
4. De q3 para q2 com "b": Representa a transição para o estado q2 (conjunto {q1}) ao encontrar "b" após "aba".

Aqui está o AFD completo em forma de tabela:

```
| Estado Atual | Próximo Estado (a) | Próximo Estado (b) |
|--------------|--------------------|--------------------|
|     q0       |       q1           |       -            |
|     q1       |       -            |       q2           |
|     q2       |       q3           |       -            |
|     q3       |       q2           |       -            |
```

Aqui está o AFD em forma de diagrama:

```
       a          b
     -----> q1 -----> q2
     |               |
     |               v
     |              q3
     |               |
     +---------------
```

Neste AFD, o estado q0 é o estado inicial, e o estado q4 é o estado final.
